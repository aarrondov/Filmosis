Package com.example.filmosis.data.model.tmdb

  • Class Summary 
    Class Description
    CastClase que representa un miembro del elenco de una producción.
    CastXClase que representa un miembro del elenco de una producción según la API de TMDB.
    CombinedCreditsClase que representa los créditos combinados de una producción, que incluyen tanto el elenco como el equipo de producción.
    CreditsCastClase que representa los créditos de un elenco de una producción, que incluyen tanto los miembros del elenco como el equipo de producción.
    CrewClase que representa un miembro del equipo de producción de una producción.
    CreditsResponseClase que representa la respuesta obtenida al solicitar información sobre el equipo de producción de una producción.
    CrewXClase que representa un miembro del equipo de producción de una producción según la API de TMDB.
    KnownForClase que representa las producciones conocidas por un miembro del elenco o equipo de producción según la API de TMDB.
    MovieClase que representa una película según la API de TMDB.
    MoviefrClase que representa una película según la API de TMDB (francés).
    IdiomaClase que representa un idioma según la API de TMDB.
    GenreClase que representa un género según la API de TMDB.
    MoviesPageClase que representa una página de resultados de películas obtenida de la API de TMDB.
    PersonClase que representa a una persona (actor, director, etc.) según la API de TMDB.
    PersonDetailsClase que representa los detalles de una persona (actor, director, etc.) según la API de TMDB.
    PersonsPageClase que representa una página de resultados de personas obtenida de la API de TMDB.
