ExploreFragment

class ExploreFragment : Fragment

Un fragmento que representa la sección de Explorar de la aplicación, permitiendo a los usuarios descubrir películas populares, buscar películas o personas, y filtrar películas según diferentes criterios. Este fragmento utiliza RecyclerViews para mostrar películas populares y listas de películas filtradas.
